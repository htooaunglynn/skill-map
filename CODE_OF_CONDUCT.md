# Code of Conduct

## Our Pledge

We want SkillMap to be a welcoming project for learners, builders, and contributors of all experience levels. We pledge to make participation in this project respectful, practical, and harassment-free.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially when discussing ideas or reviewing code.
- Keep feedback focused on the work, not the person.
- Share context when proposing changes so others can follow your reasoning.
- Respect the local-first and privacy-focused goals of the project.

## Unacceptable Behavior

- Harassment, intimidation, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing another person's private information without permission.
- Trolling, repeated disruption, or bad-faith argument.
- Pressuring maintainers or contributors for unpaid support outside normal project channels.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block users when behavior harms the project community.

If you see behavior that violates this code of conduct, report it privately to the project maintainer through GitHub.

## Scope

This code of conduct applies to project spaces including GitHub issues, pull requests, discussions, and any official project communication channels.
